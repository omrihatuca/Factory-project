In that page you have all the information about different employees and 8 option to choose, every choise will take you to another page or will make difrrent effect, the options is:

  • Edit - if you want to edit employee.
  • Delete - if you want to delete the employee, attention - delete button will delete information from related tables .
  • Add shift- if you want to add new shift for specific employee.
  • Add employee- if you want to add new employee.
  • Add new shift for new employee- if you want to add new shift for new employee.
  • Search- if you want to search employee by first name / last name / department id.
  • Home page- if you want to get back home page.
  • Log-in page- if you want to startover.
